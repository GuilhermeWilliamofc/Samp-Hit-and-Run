# Samp Hit and Run
O Hit and Run é um gamemode de Team Deathmatch para SA-MP (San Andreas Multiplayer) desenvolvido na linguagem Pawn, onde jogadores escolhem entre ser Policiais ou Ladrões através da seleção de skins padrão do jogo, competindo em uma arena próxima à Delegacia de Los Santos com armas e veículos específicos para cada equipe.
